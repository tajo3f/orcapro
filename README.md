# OrçaPro 🚀
Plataforma premium para gestão e geração de orçamentos profissionais.

## 🛠 Tecnologias
- HTML5 / CSS3 (Design Minimalista)
- JavaScript (Integração e PDF)
- [Supabase](https://supabase.com/) (Banco de Dados e Auth)
- [HTML2PDF](https://ekoopmans.github.io/html2pdf.js/) (Motor de geração de documentos)

## 📁 Estrutura de Arquivos
- `index.html`: Estrutura principal e dashboard.
- `style.css`: Estilização e UX.
- `script.js`: (Embutido no HTML no momento).

## ⚙️ Configuração
1. Crie um projeto no Supabase.
2. Rode o SQL fornecido no editor.
3. Insira suas chaves de API no arquivo principal.
